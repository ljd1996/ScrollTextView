# ScrollTextView

包含两个类型的控件。

## 高亮滚动文本

- 能够显示指定的文本(如果是 String 列表呢)，超出的内容可滚动(自动或手动)显示
- 高亮固定的行
- 能够调节字体大小，高亮字体颜色，滚动速度
